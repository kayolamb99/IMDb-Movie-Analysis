# IMDb-Movie-Analysis
Analysis of movies scraped from IMDb

## Data Source
Data obtained directly from Kaggle, which can be found [here](https://www.kaggle.com/danielgrijalvas/movies).
Contains data from over 6,000 movies from IMDb with various production-level information (director, main star),
as well as monetary results (budget, gross box office). 

Data processing can be found in the 'data_cleaning.py' file.

## Analysis

Exploratory Data Analysis (EDA) can be found in the 'eda_visualization.py' file, as well as the 'EDA.twb' file.

Full analysis and models will be uploaded
